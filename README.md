#  Rabbit â€“ E-Commerce

Rabbit Shop is a complete e-commerce platform built for a smooth online fashion shopping experience. It includes secure user authentication, an admin dashboard, product browsing, cart management, and order processing. The frontend is built with React , while the backend uses Node.js, Express & MongoDB, fully deployed on Vercel.

---

##  Live Demo

Frontend: https://rabbit-shop-h6jd.vercel.app/   

---

##  Features

###  User Features
-  Browse Latest Products
-  Filter by Category, Gender, and Best Sellers
-  Add to Cart & Update Quantity
-  Login + Register with JWT Auth
-  Place Orders
-  Subscribe for Offers

###  Admin Features
-  Admin Login
-  Add / Edit / Delete Products
-  View Manage Orders
-  Manage Users
-  Best Seller & New Arrival Tags

---

##  Tech Stack

| Part | Technology |
|------|------------|
| Frontend | React.js (Vite), Axios, React Router DOM |
| Backend | Node.js, Express.js, MongoDB |
| Authentication | JWT + Middleware-based Role Security |
| Storage | MongoDB Atlas |
| Deployment | Vercel (Frontend + Backend) |
